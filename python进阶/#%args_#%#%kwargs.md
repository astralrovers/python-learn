### 不定长参数
  - 1.理解*与**的作用  
    ```
    In [29]: s_list = (12, 34, 56)
    In [30]: var1, var2, var3 = s_list  #这个过程为元组的拆包
    In [31]: print(var1, var2, var3)
    12 34 56

    In [32]: var1, *var2 = s_list  #这里注意下,拆包的时候前面的两个参数无法对应元组中的三个元素，
                                   #在var2前面加*又相当于将s_list的剩余（var1被赋值了第一个元素）两个参数组包了
    In [33]: print(var1, var2)
    12 [34, 56]

    #更为直观的比较
    In [51]: print(s_list)
    (12, 34, 56)
    In [48]: print(*s_list) #这里传参时使用*表示拆包，
                            #s_list不再只是对应于一个变量(不加*表示这个参数是列表)而是多个,类似var1, var2, var3 = s_list，
                            #参数就相当于等号前面的变量
    12 34 56

    In [53]: def clear_arg(var1, var2):
    ...:     print(var1, var2)
    ...:     

    In [54]: c = [3, 5]

    In [55]: clear_arg(*c)
    3 5

    In [15]: def test(a, *b, **c): #这里的*、**表示组包，由于参数的个数不确定，因此不能挨个写入var1,var2这样的参数名,
                                   #因此又需要把传进来的多个参数收集到一个可迭代变量中，类似中var1, *var2 = s_list剩余的参数被var2收集了
    ...:     print(a)
    ...:     print(b)
    ...:     print(c)
    ...:
    In [22]: s_dict = {"name":"ywl"}

    In [22]: s_dict = {"name":"ywl"}
    In [23]: test(90, *s_list, **s_dict)    #*、**表示拆包
    90
    (12, 34, 56)    #结果可以看出直接打印变量是一个可迭代的变量
    {'name': 'ywl'}
    ```  
    总结就是传参时使用\*、\*\*表示拆包，函数定义时\*、\*\*表示组包。  
  - 2.\*与\*\*区别
    ```
    In [23]: test(90, *s_list, **s_dict)
    90
    (12, 34, 56) #使用*那么参数是个元组
    {'name': 'ywl'} #使用**那么参数是个字典
    ```  
    另一个栗子  
    ```
    In [52]: test(90, *s_list, sex='man')
    90
    (12, 34, 56)
    {'sex': 'man'} #貌似传参时使用的关键字就是字典的key值
    ```  
    总结就是\*表示普通参数，\*\*表示关键字参数，拆包字典就相当于key值便是关键字，与显式地使用关键字参数一样，没区别
